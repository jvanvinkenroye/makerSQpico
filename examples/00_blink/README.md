# Beispiel: Blink

Die eingebaute LED des Pico blinkt im Halbsekundentakt. Kein externes Bauteil nötig.

## Hardware

Nur der Raspberry Pi Pico – die LED ist bereits auf dem Board verbaut (`board.LED`).

## Verwendung

`code.py` in das Stammverzeichnis von `CIRCUITPY/` kopieren. Der Pico startet automatisch neu.

## Erwartetes Verhalten

Die grüne LED auf dem Pico blinkt: 0,5 s an, 0,5 s aus.
